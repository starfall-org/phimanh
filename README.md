# Phimanh

## Giới thiệu

Dự án này là một ứng dụng web được xây dựng bằng Next.js cho phép người dùng xem phim trực tuyến với giao diện thân thiện và dễ sử dụng.

## Bắt đầu

Để bắt đầu, hãy chạy server phát triển:

```bash
npm run dev
# hoặc
yarn dev
# hoặc
pnpm dev
# hoặc
bun dev
```

## Lưu ý về Pull Request
### 1. Ghi rõ mô tả về những thay đổi bạn đã thực hiện.
### 2. Vui lòng giữ nguyên tiêu đề **"Phim Ảnh"** khi gửi Pull Request để duy trì sự nhất quán trong dự án.

# Tuyên bố từ chối trách nhiệm
- Dự án này chỉ nhằm mục đích học tập và phát triển kỹ năng lập trình.
- Chúng tôi không lưu trữ hoặc phân phối bất kỳ nội dung vi phạm bản quyền nào.
- Vui lòng sử dụng ứng dụng này một cách hợp pháp và tôn trọng quyền sở hữu trí tuệ.
- Chúng tôi không chịu trách nhiệm về bất kỳ hành vi vi phạm pháp luật nào phát sinh từ việc sử dụng ứng dụng này.
